# ArtCommission API

Platform manajemen pesanan karya seni digital.

## Fitur

- Manajemen user (artist/client)
- Sistem portfolio
- Manajemen pesanan
- Progress tracking
- Sistem pembayaran
- Review & rating

## Tech Stack

- FastAPI
- PostgreSQL
- SQLAlchemy
- Pydantic
- JWT Authentication
